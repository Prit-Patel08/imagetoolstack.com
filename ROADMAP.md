## Monetization Strategy

### Phase 1: Ad Revenue
- **Placement A:** Below tool interface, above content (728×90 leaderboard)
- **Placement B:** Sidebar sticky (300×250 rectangle)
- **Rule:** NEVER inside the tool workflow. Ads appear only in content zones.

### Phase 2: Premium Features (Future)
- Batch processing (100+ images)
- API access for developers
- Custom watermark templates
- Priority processing queue
- Remove ads experience

### Phase 3: Enterprise
- White-label tool embedding
- Team collaboration features
- Usage analytics dashboard

├── /terms
